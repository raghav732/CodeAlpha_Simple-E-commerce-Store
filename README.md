# CodeAlpha Simple E-Commerce Store

## Overview
A simple E-Commerce website developed using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and Mongoose.

## Features
- User Registration
- User Login
- Logout Functionality
- Product Listing
- Product Details Page
- Shopping Cart
- Add to Cart
- Remove from Cart
- Order Processing
- MongoDB Database Integration

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Database Collections
- Users
- Products
- Orders

## Installation

```bash
npm install
node server.js
```

## Run Project

Open:

```text
http://localhost:3000
```
